NetKet

Machine Learning for Many-Body Quantum Systems

Get Started

What is NetKet?

NetKet is an open-source project delivering cutting-edge methods for the study of many-body quantum systems.

Neural-Network Quantum States and more

NetKet provides a framework to implement and use the most advanced techniques to study many-body quantum systems with artificial-neural networks and machine learning techniques.

Modular Styling and Templating

NetKet provides a modular infrastructure for applications of machine-learning techniques to many-body quantum systems. For example, you can easily set up your own machine and take advantage of all of the other tools available in NetKet.
