    netket.machine.Jastrow¶

    class netket.machine.Jastrow¶

    A Jastrow wavefunction Machine. This machine defines the following wavefunction:

    \[\Psi(s_1,\dots s_N) = e^{\sum_{ij} s_i W_{ij} s_j}\]

    where :math:` W_{ij}` are the Jastrow parameters.

    __init__(self: netket._C_netket.machine.Jastrow, hilbert: netket._C_netket.hilbert.Hilbert) → None¶

    Constructs a new Jastrow machine:

    Parameters

    hilbert – Hilbert space object for the system.

    Examples

    A Jastrow machine for a one-dimensional L=20 spin 1/2 system:

    >>> from netket.machine import Jastrow
>>> from netket.hilbert import Spin
>>> from netket.graph import Hypercube
>>> g = Hypercube(length=20, n_dim=1)
>>> hi = Spin(s=0.5, total_sz=0, graph=g)
>>> ma = Jastrow(hilbert=hi)
>>> print(ma.n_par)
190

    to_array(self: netket._C_netket.machine.Machine, normalize: bool = True) → numpy.ndarray[complex128[m, 1]]¶

    Returns a numpy array representation of the machine. The returned array is normalized to 1 in L2 norm. Note that, in general, the size of the array is exponential in the number of quantum numbers, and this operation should thus only be performed for low-dimensional Hilbert spaces.

    This method requires an indexable Hilbert space.
